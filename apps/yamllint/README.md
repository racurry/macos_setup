# yamllint

YAML file linter for style consistency and syntax validation.

## Installation

```bash
brew install yamllint
```

## Setup

```bash
./apps/yamllint/yamllint.sh setup
```

This symlinks the config to `~/.config/yamllint/config`.

## Syncing Preferences

Repo sync. Config symlinked to `~/.config/yamllint/`.
