# macOS

macOS system preferences and folder structure configuration.

## Contents

- `folders.sh` - Script to create standard folder structure
- `macos.sh` - Script to configure macOS system preferences
- `macos_screenshot_settings.png` - Reference image for screenshot settings

## Setup

Run `./apps/macos/macos.sh` to set up system preferences.
Run `./apps/macos/folders.sh` to create folder structure.
