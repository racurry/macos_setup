# OpenSCAD

3D modeling via code.

## Setup

```bash
./scripts/bash/openscad.sh
```

## Usage

**VS Code:**
1. Open `.scad` file
2. Click "Preview in OpenSCAD" (top right)
3. Edit & save - preview auto-reloads

**Command-line:**
```bash
openscad -o output.stl input.scad
openscad -o output.stl -D 'param=value' input.scad
```
